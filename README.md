## 🚀 Project Overview
An **AI-powered House Price Prediction System** that uses Machine Learning to estimate property prices based on:

- 📏 Area (sqft)
- 🛏️ Bedrooms
- 🚿 Bathrooms
- 📍 Location
- 🏢 Other housing features

---

## 🎯 Goal
To build a **real-world regression model** that helps predict house prices accurately using historical housing data.
